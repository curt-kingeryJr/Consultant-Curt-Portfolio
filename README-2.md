## 💼 Featured Projects

### 🟩 [Open Orders Report](./open-orders-project/)

Power BI dashboard built to track order aging, lead times, and vendor
performance. Resulted in \$1M+ time savings by automating previously
manual reporting.

### 🟦 [Healthcare Billing Analysis](./healthcare-billing-analysis/)

Excel-based analysis of healthcare billing data with custom
transformations and time series exploration. Applied the **Pareto
Principle** to identify key cost drivers.

### 🟨 [Insurance Claims Analysis](./Insurance_Claims_Analysis/)

Applied root cause and Pareto-driven analysis to insurance claims data.
Highlighted the small set of variables (e.g., smoker status, BMI)
driving the majority of claim costs.

### 🟪 [Freshman GPA Prediction](./freshman-gpa-prediction/)

Predictive analytics project using **linear regression** to model and
forecast GPA outcomes for first-year students. Demonstrates applied
**machine learning methods** for prediction.
